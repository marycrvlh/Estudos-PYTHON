# Estudos de Python
Este repositório foi criado com o objetivo de registrar minha jornada de aprendizado na linguagem Python, reunindo exercícios, testes e pequenos projetos desenvolvidos durante meus estudos.
Alguns dos códigos presentes aqui foram desenvolvidos durante as aulas do curso de Desenvolvimento de Sistemas, como parte das atividades práticas de programação e lógica.

# Objetivo
Praticar programação e acompanhar minha evolução em Python, explorando conceitos como:
- Lógica de programação
- Variáveis e tipos de dados
- Estruturas condicionais (if, else, elif)
- Estruturas de repetição (for, while)
- Funções
- Listas, tuplas e dicionários
- Pequenos programas e exercícios práticos
